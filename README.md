# JEEVO

> Comprehensive healthcare technology platform — "Jeevo" delivers connected patient experiences, provider tools, and administrative workflows.

## Summary
- **Purpose:** A modern, React + TypeScript frontend for a healthcare platform with patient portal, provider search, appointments, and AI-powered tools.
- **Status:** Initial commit — project scaffold and components present.

## Features
- Patient portal UI
- Appointment booking
- Image analysis and AI integrations
- Trust, compliance, and legal pages

## Tech Stack
- React + TypeScript
- Vite for build tooling
- Node.js / npm for package management

## Quick Start (Development)
1. Install Node.js (16+ recommended) and npm.
2. Install dependencies:

```powershell
npm install
```

3. Start the dev server:

```powershell
npm run dev
```

4. Open http://localhost:5173 (or the address printed by Vite).

## Build

```powershell
npm run build
```
